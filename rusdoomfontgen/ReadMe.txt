========================================================================
    rusdoomfontgen
========================================================================

Интерактивное приложения для создания шрифтов типа FON2 для ZDoom (http://zdoom.org)
Итоговый файл со шрифтом носит название "dbigfont.fon2". Все права на созданный шрифт остаются за праводержателем оригинальных шрифтов (обычно это корпорация Microsoft), если в законодательстве вашей страны не указано обратное.
Исходный код выдаётся по требованию.

Дневник изменений.
С версии 1.0:
+Добавлена возможность задать список символов, которые будут записаны в шрифт. Символы беруться из файла chars.txt. Если файла нет, будет использоваться стандартный набор.
*Исправлена сортировка символов.
*Исправлена потеря одной колонки пикселей в символе.
+Добавлен диалог составления фильтров.
+Добавлен фильтр обводки границы.
*Исправлено преобразование в оттенки серого.
+Добавлен фильтр порога.
+Добавлены условия к фильтрам.

С версии 2.0
+Добавлена поддержка кодовых страниц
+Добавлена поддержка италика
+Добавлена поддержка кернинга
